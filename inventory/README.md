## Règle

Les références doivent être vérifiées sur le terrain ou chez le fournisseur avant d'être considérées comme validées.

Ne pas confondre :

- la désignation interne ;
- la référence fournisseur ;
- la référence fabricant ;
- le diamètre réel du tuyau ou du filetage.
