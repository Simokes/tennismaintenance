# STOCK-001 — Stock arrosage

**Statut :** À compléter  
**Version :** 0.1  
**Date :** 2026-07-09

## 1. Objectif

Définir le stock utile pour maintenir les sorties d'eau terrain, les sorties d'entretien et les ensembles d'arrosage manuel.

Le stock doit permettre de réparer rapidement les pannes courantes sans attendre une commande.

## 2. Règle de gestion

Pour chaque pièce critique, conserver :

- une quantité minimale ;
- une référence claire ;
- un fournisseur identifié ;
- une compatibilité avec les standards concernés.
