# Décisions techniques

Les décisions techniques expliquent pourquoi un choix a été fait.

Elles évitent de rediscuter les mêmes sujets plusieurs années plus tard.

## Format ADR

Chaque décision doit contenir :

- contexte ;
- problème ;
- options étudiées ;
- décision ;
- conséquences ;
- statut ;
- date.
