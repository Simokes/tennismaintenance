# ADR-0002 — Utiliser GEKA pour les sorties d'entretien

## Statut
Validé

## Contexte
Les sorties d'entretien doivent alimenter différents équipements (Kärcher, tuyaux Ø19, Ø25, nettoyage...).

## Décision
Standardiser les sorties d'entretien avec un raccord GEKA.

## Justification
- connexion rapide ;
- standard robuste ;
- un seul embout fixe sur la sortie ;
- chaque équipement possède son propre raccord GEKA.

## Conséquences
Les agents n'ont plus à modifier la sortie d'eau lors des changements d'équipement. Seuls les équipements mobiles changent.
