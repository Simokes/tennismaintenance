# Procédures

Les procédures décrivent les gestes d'intervention.

Une procédure répond à la question : **comment réaliser correctement une opération de maintenance ?**

## Procédures

| Référence | Sujet | Statut |
|---|---|---|
| [PROC-001](./PROC-001-monter-sortie-terrain-standard.md) | Monter une sortie terrain standard | Brouillon |
| [PROC-002](./PROC-002-remplacer-joint-geka.md) | Remplacer un joint GEKA | Brouillon |
| [PROC-003](./PROC-003-diagnostiquer-sortie-eau-faible-debit.md) | Diagnostiquer une sortie d'eau à faible débit | Brouillon |
| [PROC-004](./PROC-004-fabriquer-buse-cuivre.md) | Fabriquer une buse cuivre standard | Brouillon |

## Format recommandé

Chaque procédure doit contenir :

- objectif ;
- périmètre ;
- matériel nécessaire ;
- EPI ;
- étapes ;
- contrôles finaux ;
- observations terrain ;
- historique.

## Règle

Une procédure doit pouvoir être utilisée par un agent sur le terrain.

Si elle devient trop théorique, elle doit être simplifiée.
