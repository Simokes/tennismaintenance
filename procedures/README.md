# Procédures

Les procédures décrivent les gestes d'intervention.

Une procédure répond à la question : **comment réaliser correctement une opération de maintenance ?**

## Format recommandé

Chaque procédure doit contenir :

- objectif ;
- périmètre ;
- matériel nécessaire ;
- EPI ;
- étapes ;
- contrôles finaux ;
- observations terrain ;
- historique.
