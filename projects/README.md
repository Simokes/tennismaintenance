# Projets d'amélioration

Ce domaine regroupe les dossiers d'aide à la décision portant sur l'amélioration durable des installations, des équipements et des méthodes de maintenance.

## Rôle

Un projet décrit :

- le problème observé ;
- les objectifs recherchés ;
- les solutions étudiées ;
- les contraintes et risques ;
- le budget à confirmer ;
- l'essai ou la décision proposée ;
- le résultat après mise en œuvre.

Les projets ne remplacent ni les interventions ni les décisions d'architecture :

- `interventions/` conserve les opérations réalisées ;
- `decisions/` conserve les décisions structurantes ;
- `projects/` prépare, compare et suit les améliorations proposées.
