# Interventions

Cette section conserve les historiques d'intervention importants.

Elle ne remplace pas un outil officiel de suivi d'activité. Elle sert à capitaliser les interventions techniques utiles au référentiel et à fournir un support papier simple pour les interventions terrain.

## Objectif

Conserver la mémoire des opérations qui expliquent l'état réel du site :

- remplacement d'une sortie d'eau ;
- modification d'un raccord ;
- changement de standard ;
- panne récurrente ;
- réparation importante ;
- observation terrain utile.

## Articulation des supports

- utiliser la fiche papier pour le constat, les travaux réalisés et le visa du responsable ;
- reporter dans `INT-2026` les interventions qui doivent rester dans l'historique ;
- créer une fiche détaillée à partir de `templates/INT-template` uniquement pour une intervention importante ou réutilisable comme référence.

## Règle

Toutes les petites opérations quotidiennes n'ont pas besoin d'être documentées dans Git.

Documenter seulement ce qui sera utile dans 6 mois ou dans 5 ans.
