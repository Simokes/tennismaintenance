# Icônes

Bibliothèque de symboles normalisés pour les plans de maintenance du réseau d'arrosage.

## Fichier

- `bibliotheque-icones-arrosage.svg` — planche complète (A4 paysage), imprimable
  et éditable sous Inkscape.

## Catégories

| Section | Icônes |
|---------|--------|
| Points du plan | AR (arroseur), REG (regard), SET (sortie d'eau), ACC (accès / portillon) |
| Composants hydrauliques | EV (électrovanne), VM (vanne manuelle), RP (réducteur de pression), FIL (filtre), CAR (clapet anti-retour), PRG (purge / vidange), CPT (compteur), PMP (pompe), RES (réserve / bâche) |
| Réseau / canalisations | canalisation principale, canalisation secondaire, TE (dérivation), CD (coude), BCH (bouchon / fin de ligne) |
| Commande & électrique | PROG (programmateur), CE (coffret électrique), CPL (capteur de pluie), TR (transformateur) |
| Repères & annotations | Nord, court de tennis, échelle |

## Palette

- Bleu `#1a4b9e` — eau / arroseurs
- Vert `#1a7a3a` — regards
- Rouge `#c0392b` — électrique / commande
- Orange `#d97a1a` — surface (terre battue)

Palette identique à `assets/plans/Template_plan-arrosage.svg` pour que les
symboles reportés dans la légende d'un plan restent cohérents.
