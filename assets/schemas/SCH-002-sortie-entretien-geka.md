# SCH-002 — Sortie d'entretien GEKA

**Statut :** À dessiner  
**Version :** 0.1  
**Date :** 2026-07-09  
**Documents liés :** STD-002, EQ-003, PROC-002

## 1. Objectif

Créer un schéma clair de la sortie d'entretien standard équipée en GEKA.

## 2. Contenu attendu

```text
Sortie réseau
        ↓
Robinet de puisage standard
        ↓
Raccord GEKA fixe
        ↓
Tuyau ou équipement mobile compatible GEKA
```

## 3. Repères à afficher

| Repère | Désignation |
|---|---|
| SR | Sortie réseau |
| ROB | Robinet de puisage standard |
| GKF | Raccord GEKA fixe |
| JGK | Joint GEKA remplaçable |
| TM | Tuyau / équipement mobile |

## 4. Contraintes du dessin

- Montrer que le GEKA est l'interface commune.
- Montrer plusieurs équipements mobiles possibles : tuyau Ø19, tuyau Ø25, Kärcher.
- Indiquer que le joint GEKA est remplaçable.
- Ne pas représenter cette sortie comme une sortie terrain.
