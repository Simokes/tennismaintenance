# SCH-004 — Buse cuivre custom

**Statut :** À dessiner  
**Version :** 0.1  
**Date :** 2026-07-09  
**Documents liés :** STD-004, EQ-004, PROC-004

## 1. Objectif

Créer un schéma de la buse cuivre custom utilisée pour obtenir un jet de type pluie.

## 2. Contenu attendu

```text
Raccord 1" femelle → cannelé Ø25
        ↓
Soudure étain
        ↓
Tube cuivre Ø25
        ↓
Embout aplati
        ↓
Jet pluie
```

## 3. Repères à afficher

| Repère | Désignation |
|---|---|
| RC3 | Raccord 1" femelle → cannelé Ø25 |
| S | Soudure étain |
| TC | Tube cuivre Ø25 |
| EMB | Embout aplati |
| JP | Jet pluie |

## 4. Contraintes du dessin

- Montrer la buse comme une pièce custom fabriquée en interne.
- Montrer l'embout aplati.
- Montrer la soudure.
- Ajouter les mesures à compléter : longueur totale, largeur aplatie, ouverture de sortie.
