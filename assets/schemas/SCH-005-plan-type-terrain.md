# SCH-005 — Plan type terrain avec 6 arroseurs

**Statut :** À dessiner  
**Version :** 0.1  
**Date :** 2026-07-09  
**Documents liés :** FIC-T01 à FIC-T06

## 1. Objectif

Créer un plan type de terrain permettant de repérer les 6 arroseurs fixes, la sortie d'eau terrain et les regards utiles au diagnostic.

## 2. Contenu attendu

```text
Fond nord

      A1          A2


A3                    A4


      A5          A6

Fond sud
```

## 3. Repères à afficher

| Repère | Désignation |
|---|---|
| A1 à A6 | Arroseurs fixes |
| SE | Sortie d'eau terrain |
| R1, R2... | Regards utiles au diagnostic |
| Accès | Accès terrain si utile |
| Nord | Orientation approximative |

## 4. Contraintes du dessin

- Le plan doit rester générique.
- Les fiches terrain T1 à T6 décriront les variations réelles.
- Les regards ne doivent pas être décrits techniquement en détail.
- Seule leur fonction de diagnostic est représentée.
