# Schémas

Cette section contient les schémas techniques du référentiel.

## Objectif

Les schémas servent à rendre les standards lisibles rapidement sur le terrain.

Ils doivent être :

- simples ;
- fidèles ;
- imprimables en A4 ;
- compréhensibles sans explication orale ;
- liés aux standards et procédures concernés.

## Formats recommandés

| Format | Usage |
|---|---|
| `.drawio` | Source modifiable du schéma |
| `.png` | Export image pour GitHub / impression |
| `.svg` | Export vectoriel si utile |
| `.md` | Description textuelle du schéma |

## Règle

Un schéma ne doit pas inventer la réalité terrain.

Quand une information n'est pas encore confirmée, le schéma doit indiquer `À confirmer` ou rester générique.
