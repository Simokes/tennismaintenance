# Croquis

Cette section contient les croquis rapides réalisés avant les schémas propres.

## Objectif

Les croquis servent à capturer rapidement la réalité terrain.

Ils peuvent être :

- faits à la main ;
- photographiés ;
- remplacés plus tard par un schéma propre ;
- conservés pour historique.

## Usage recommandé

Utiliser les croquis pour :

- positionner les regards ;
- noter les sorties d'eau ;
- repérer les arroseurs ;
- dessiner les différences entre terrains ;
- préparer les schémas `draw.io`.

## Nommage recommandé

```text
CROQ-T01-plan-arrosage-2026-07-09.jpg
CROQ-site-general-2026-07-09.jpg
CROQ-sortie-terrain-standard-2026-07-09.jpg
```

## Règle

Un croquis n'a pas besoin d'être beau.

Il doit être compréhensible et daté.
