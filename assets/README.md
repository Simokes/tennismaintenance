# Assets

Cette section contient les schémas, photos et supports visuels.

## Organisation recommandée

```text
assets/
├── schemas/
├── photos/
└── croquis/
```

Les photos doivent être nommées de façon explicite, par exemple :

```text
T01-sortie-eau-2026-07-09.jpg
STD-001-montage-sortie-terrain.png
```
