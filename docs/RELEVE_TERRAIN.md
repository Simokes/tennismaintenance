# Relevé terrain — checklist

## Objectif

Cette checklist sert à remplir progressivement les fiches `SITE-001` et `FIC-T01` à `FIC-T06`.

Le but est de noter uniquement ce qui est observé ou vérifié sur place.

## Règle

Ne pas deviner.

Si une information n'est pas certaine, écrire :

```text
À confirmer
```

ou

```text
À relever
```

## 1. Relevé général du site

- [ ] Nom officiel du site
- [ ] Adresse
- [ ] Nombre exact de terrains
- [ ] Position des vestiaires
- [ ] Position du local technique
- [ ] Position des sorties d'eau d'entretien
- [ ] Position des zones de stockage
- [ ] Photo générale du complexe

## 2. Pour chaque terrain

À répéter pour T1 à T6.

### Identification

- [ ] Numéro du terrain visible
- [ ] Orientation approximative nord/sud
- [ ] Accès principal
- [ ] Particularités visibles

### Sortie d'eau terrain

- [ ] Position de la sortie d'eau
- [ ] Type de sortie actuelle
- [ ] Présence d'une vanne primaire
- [ ] Type de raccord actuel
- [ ] État général
- [ ] Photo de la sortie

### Arroseurs fixes

- [ ] Confirmer qu'il y a 6 arroseurs
- [ ] Repérer A1 à A6 sur le croquis
- [ ] Vérifier modèle si visible
- [ ] Noter les arroseurs faibles, bouchés ou mal orientés

### Regards utiles au diagnostic

- [ ] Repérer les regards accessibles
- [ ] Noter leur localisation
- [ ] Noter ce qu'ils semblent commander, uniquement si vérifié
- [ ] Noter la présence de vanne / électrovanne si visible sans démontage technique
- [ ] Photo de chaque regard fermé

## 3. Sorties d'entretien hors terrain

Pour chaque sortie hors terrain :

- [ ] donner un repère `SE-XX` ;
- [ ] noter la localisation ;
- [ ] noter le type de robinet actuel ;
- [ ] noter le type de raccord actuel ;
- [ ] vérifier si passage vers STD-002 possible ;
- [ ] photo.

## 4. Équipements mobiles

- [ ] nombre de tuyaux Ø25 terrain
- [ ] longueur approximative des tuyaux terrain
- [ ] nombre de buses cuivre
- [ ] état des buses cuivre
- [ ] nombre de tuyaux entretien Ø19 ou Ø25
- [ ] accessoires Kärcher
- [ ] raccords GEKA disponibles

## 5. Photos minimales

Pour chaque terrain :

- [ ] vue générale ;
- [ ] sortie d'eau ;
- [ ] buse / tuyau utilisé ;
- [ ] chaque regard utile ;
- [ ] défaut notable.

## 6. Première passe recommandée

Ne pas chercher à tout faire parfaitement.

Objectif de la première passe :

1. localiser les sorties d'eau ;
2. prendre les photos ;
3. repérer les regards ;
4. noter les différences entre les terrains.

La précision viendra ensuite.
