# STD-002 — Sortie d'eau d'entretien

## Usage

Sortie hors terrain pour les usages polyvalents : nettoyage, Kärcher, remplissage, arrosage hors terrain.

Priorités :

- réglage progressif du débit ;
- connexion rapide ;
- compatibilité GEKA ;
- remplacement facile des joints.

## Configuration cible

```text
Sortie réseau
        ↓
Robinet de puisage standard
        ↓
Raccord GEKA vissé
        ↓
Tuyau ou équipement mobile compatible GEKA
```

## Repères

| Repère | Désignation |
|---|---|
| SR | Sortie réseau |
| ROB | Robinet de puisage |
| GKF | Raccord GEKA fixe |
| JGK | Joint GEKA remplaçable |
