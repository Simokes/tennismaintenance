# STD-003 — Ensemble d'arrosage manuel

## Usage

Ensemble mobile manipulé par l'agent pour arroser la terre battue avec un jet de type pluie.

## Configuration cible

```text
Tuyau souple Ø25
        ↓
Raccord cannelé Ø25 → femelle 1"
        ↓
Vanne secondaire 1/4 tour 1" M/M
        ↓
Raccord 1" femelle → cannelé Ø25
        ↓
Buse cuivre custom Ø25
```

## Point clé

La vanne secondaire permet de couper l'eau directement côté poignée sans retourner à la sortie réseau.
