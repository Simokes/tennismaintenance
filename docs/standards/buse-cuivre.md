# STD-004 — Buse cuivre standard

## Usage

Buse custom utilisée à l'extrémité de l'ensemble d'arrosage manuel pour obtenir un jet de type pluie.

## Configuration cible

```text
Raccord 1" femelle → cannelé Ø25
        ↓
Soudure étain
        ↓
Tube cuivre Ø25
        ↓
Extrémité aplatie
        ↓
Jet pluie
```

## Critères de conformité

- jet diffus de type pluie ;
- pas de jet concentré qui creuse la terre battue ;
- pas de fuite à la soudure ;
- pas d'arête coupante ;
- compatible avec l'ensemble STD-003.
