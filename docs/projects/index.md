## Principe

Un projet ne vaut pas décision automatique. Il sert à :

- objectiver un problème ;
- comparer plusieurs solutions ;
- chiffrer les options ;
- proposer un essai limité ;
- recueillir une validation hiérarchique ;
- conserver le retour d'expérience.
