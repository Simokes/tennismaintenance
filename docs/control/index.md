# Commande et programmation

Cette section documente les équipements qui commandent l'arrosage du complexe.

Elle complète les fiches terrain :

- les fiches terrain indiquent **où se trouvent les arroseurs** ;
- les fiches de commande indiquent **quel canal les déclenche** ;
- les programmes indiquent **quand et pendant combien de temps ils fonctionnent**.

## Équipements documentés

- [CTRL-001 — Programmateur Hunter ICC](programmateur-hunter-icc.md)

## Règle de relevé

Une correspondance canal → arroseurs ne doit être marquée comme vérifiée qu'après un essai manuel ou une source technique fiable.
