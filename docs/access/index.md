# Accès

Cette section documente le fonctionnement et la maintenance courante des accès à code des deux sites.

## Documents disponibles

- [La Pastorale — claviers, ventouses et grooms](controle-acces-site.md)
- [Le Kalisté — claviers sur piles, gâches et cadenas du parking](controle-acces-kaliste.md)

## Différences entre les sites

| Site | Commande | Verrouillage | Fermeture / particularité |
|---|---|---|---|
| La Pastorale | Clavier à code alimenté électriquement | Ventouse électromagnétique | Groom |
| Le Kalisté | Clavier à code alimenté par piles | Gâche électrique | Ferme-porte éventuel à relever |
| Parking du Kalisté | Cadenas à code | Mécanique | Modèle à relever |

## Principe de sécurité

Le dépôt décrit les équipements, les procédures et les contrôles à effectuer. Il ne contient aucun code d’accès, code maître, mot de passe, combinaison ou autre secret.

Les codes en vigueur sont conservés physiquement au bureau.
