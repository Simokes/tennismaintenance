# STD-002 — Sortie d'eau d'entretien

**Statut :** Brouillon  
**Version :** 0.1  
**Date :** 2026-07-09  
**Famille :** Point d'eau hors terrain

## 1. Objectif

Standardiser les sorties d'eau hors terrain utilisées par les agents pour l'entretien général du complexe.

Ces sorties doivent être polyvalentes, faciles à utiliser et compatibles avec plusieurs équipements mobiles.

## 2. Usages prévus

- Kärcher ;
- nettoyage du matériel ;
- remplissage de seaux ;
- arrosage des zones annexes ;
- lavage ponctuel ;
- entretien hors terrain.

## 3. Configuration cible

```text
Sortie réseau
        ↓
Robinet de puisage standard
        ↓
Raccord GEKA vissé
        ↓
Tuyau ou équipement mobile compatible GEKA
```

## 4. Principe

Contrairement aux sorties terrain, les sorties d'entretien ne doivent pas utiliser une simple vanne 1/4 de tour comme organe principal.

Elles utilisent un robinet de puisage classique afin de permettre :

- un réglage progressif du débit ;
- une ouverture plus douce ;
- une meilleure polyvalence ;
- une utilisation plus adaptée aux petits tuyaux et au Kärcher.

## 5. Interface GEKA

Le raccord GEKA est utilisé comme interface commune pour les équipements mobiles.

Principe :

- la sortie d'eau reste fixe ;
- les tuyaux et accessoires portent chacun leur embout GEKA adapté ;
- l'agent ne change pas l'embout de sortie du robinet.

Exemples :

| Équipement mobile | Embout côté tuyau |
|---|---|
| Tuyau Ø19 pour Kärcher | GEKA → cannelé Ø19 |
| Tuyau Ø25 entretien | GEKA → cannelé Ø25 |
| Lance d'arrosage | GEKA adapté |

## 6. Maintenance

Le raccord GEKA doit être choisi de préférence en laiton.

Le joint GEKA est une pièce d'usure remplaçable.

Stock recommandé :

- joints GEKA EPDM ;
- raccords GEKA de rechange ;
- embouts GEKA cannelés Ø19 ;
- embouts GEKA cannelés Ø25.
