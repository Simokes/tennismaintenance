# STD-004 — Buse cuivre standard

**Statut :** Brouillon  
**Version :** 0.1  
**Date :** 2026-07-09  
**Famille :** Pièce custom terrain

## 1. Objectif

Documenter la buse cuivre utilisée pour l'arrosage manuel des terrains en terre battue.

Cette buse est une pièce custom fabriquée à partir d'un tube cuivre Ø25 dont l'extrémité est façonnée à plat afin d'obtenir un jet de type pluie.

## 2. Description

La buse est composée de :

- un tube cuivre Ø25 ;
- une extrémité aplatie et façonnée ;
- un raccord 1" femelle → cannelé ;
- une liaison soudée à l'étain entre le raccord cannelé et le tube cuivre.

## 3. Fonction recherchée

La buse doit produire un jet doux de type pluie, adapté à la terre battue.

Objectifs :

- éviter un jet trop concentré ;
- limiter le creusement du terrain ;
- permettre un arrosage manuel homogène ;
- conserver un bon débit.

## 4. Configuration

```text
Vanne secondaire 1" M/M
        ↓
Raccord 1" femelle → cannelé
        ↓
Soudure étain
        ↓
Tube cuivre Ø25
        ↓
Extrémité aplatie formant jet pluie
```
