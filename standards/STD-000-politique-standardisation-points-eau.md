# STD-000 — Politique de standardisation des points d'eau

**Statut :** Brouillon  
**Version :** 0.1  
**Date :** 2026-07-09  
**Périmètre :** Complexe municipal de tennis

## 1. Objectif

Définir une politique simple pour standardiser les points d'eau utilisés par les agents terrain.

Objectifs principaux :

- faciliter la maintenance ;
- réduire le nombre de pièces différentes ;
- clarifier les usages ;
- améliorer l'ergonomie des agents ;
- rendre les installations plus faciles à dépanner ;
- conserver une trace des choix techniques.

## 2. Périmètre

Ce standard concerne :

- les sorties d'eau des terrains ;
- les sorties d'eau hors terrain utilisées pour l'entretien ;
- les tuyaux et accessoires mobiles associés ;
- les raccords, vannes, buses et robinets accessibles aux agents terrain.

Ce standard ne concerne pas :

- le réseau d'alimentation général ;
- les pompes ;
- la programmation centrale ;
- le réseau enterré amont ;
- les éléments gérés par un autre service.

Les éléments hors périmètre peuvent être mentionnés uniquement lorsqu'ils sont utiles au diagnostic terrain.

## 3. Principe général

Les points d'eau sont standardisés selon leur usage réel.

Il existe deux familles principales :

| Famille | Usage | Philosophie |
|---|---|---|
| STD-001 | Sorties terrain | Débit, robustesse, ouverture/fermeture rapide |
| STD-002 | Sorties entretien | Polyvalence, réglage du débit, connexion rapide |

## 4. Sorties terrain

Les sorties terrain servent principalement à l'arrosage manuel de la terre battue.

Elles doivent privilégier :

- le débit ;
- la robustesse ;
- la simplicité mécanique ;
- la rapidité d'ouverture et de fermeture ;
- la compatibilité avec l'ensemble d'arrosage manuel standard.

Elles utilisent une vanne 1/4 de tour.

## 5. Sorties entretien

Les sorties entretien servent aux usages polyvalents :

- Kärcher ;
- nettoyage ;
- remplissage ;
- arrosage de zones annexes ;
- entretien général.

Elles doivent privilégier :

- le réglage progressif du débit ;
- la compatibilité avec plusieurs tuyaux ;
- le remplacement facile des accessoires ;
- la simplicité pour les agents.

Elles utilisent un robinet de puisage classique et un raccord GEKA.

## 6. Règle de conception

L'uniformisation ne doit pas être appliquée aveuglément.

Le bon standard est celui qui correspond au bon usage :

- vanne 1/4 de tour pour les sorties terrain ;
- robinet de puisage pour les sorties d'entretien ;
- GEKA pour les connexions polyvalentes hors terrain ;
- montage vissé/cannelé robuste pour l'arrosage manuel des terrains.

## 7. Évolutions

Toute évolution importante doit être documentée dans un ADR.

Exemples :

- changement de type de raccord ;
- passage à un nouveau diamètre de tuyau ;
- remplacement du standard GEKA ;
- modification de la buse cuivre ;
- suppression ou ajout d'une vanne.
