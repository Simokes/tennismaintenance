# Équipements

Cette section décrit les équipements manipulés ou maintenus par les agents.

Contrairement aux standards, qui décrivent une configuration cible, les fiches `EQ` décrivent des ensembles fonctionnels utilisés sur le terrain.

## Règle

Une fiche équipement doit répondre aux questions pratiques :

- à quoi sert l'équipement ;
- où il est utilisé ;
- à quel standard il correspond ;
- quelles pièces le composent ;
- comment le contrôler ;
- quelles pannes sont fréquentes.
